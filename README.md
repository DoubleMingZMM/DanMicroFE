# DanMicroFE

This is an example of examples on how to use [single-spa] in microfrontend.

- React
- redux
- react-router
- Webpack
- Babel
- Vue
- vueX
- vue-router

## How to get the examples running locally
```bash
git clone 
cd DanMicroFE

npm start
```
